# 做了么

一个极简的每日行动确认页。它只问一件事：今天该做的那件事，你做了么？

在线体验：[doleme.vercel.app](https://doleme.vercel.app)

## 这个项目是什么

`做了么` 是一个轻量网页，不做复杂计划管理，也不记录长篇复盘。它把行动检查压缩成一个按钮：做了，就计数；没做，就给你一点直接反馈。

适合用来做：

- 每天学习、运动、写作、项目推进的打卡入口
- 群里互相提醒的轻量链接
- 给拖延状态一个即时反馈的小工具
- 单页 HTML 项目的部署练习

## 已有功能

- **一键打卡**：点击“做了”后累计本地次数
- **本地保存**：使用浏览器 `localStorage` 保存计数，不需要账号
- **即时反馈**：点击“还没”时给出随机提醒文案
- **分享文案**：复制当前链接和累计次数，方便发给朋友或群聊
- **打赏弹窗**：内置微信/支付宝图片位，用于个人项目支持

## 本地打开

这个项目不需要构建工具，直接打开即可：

```bash
start index.html
```

也可以用任意静态服务运行：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 部署

适合直接部署到 Vercel、GitHub Pages、Netlify 或任何静态托管服务。仓库根目录就是站点根目录。

## 说明

这个项目的重点不是完整任务管理，而是用最短路径完成一次行动确认。它适合做“今天有没有动”的入口，不适合替代长期项目管理工具。
